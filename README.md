# LevelSketch
LevelSketch is a generic 3D level editor built in rust using the [bevy](https://bevyengine.org/) game engine. The aim of this tool is to provide a quick and simple 3D editor for mocking up levels and export them to various engines, both common and custom.

The latest version is [0.1.0](https://https://github.com/mdavisprog/LevelSketch/wiki/0.1.0). This page contains more information about this release.
