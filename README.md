# LevelSketch
LevelSketch is a general purpose 2D/3D level editor built in Rust using the [bevy](https://bevyengine.org/) game engine. The aim of this tool is to provide a quick and simple editor for mocking up levels and export them to various engines, both common and custom. The editor makes use of the language server protocol which determines the types that can be added to an entity.
